PICT B.E. Course Structure

First Year (FE)

Core subjects for the first year, focusing on basic engineering principles.

Subjects

  • Mathematics I - Fundamentals of Calculus and Linear Algebra
  • Physics - Concepts of Mechanics and Optics
  • Computer Programming - Introduction to C Programming
  • Electrical Engineering - Basic Electrical Principles and Circuits

Second Year (SE)

Subjects focusing on expanding knowledge in core engineering fields.

Subjects

  • Mathematics II - Advanced Calculus and Differential Equations
  • Data Structures - Introduction to Data Structures and Algorithms
  • Digital Electronics - Basics of Digital Logic Design
  • Mechanical Engineering - Statics and Dynamics of Machines

Third Year (TE)

Focus on specialized areas like software engineering and electronics.

Subjects

  • Operating Systems - Concepts and Design of Operating Systems
  • Database Management - Introduction to DBMS and SQL
  • Computer Networks - Principles of Networking and Protocols
  • Microprocessor - Architecture and Programming of Microprocessors

Final Year (BE)

Advanced subjects to prepare for real-world applications and projects.

Subjects

  • Software Engineering - Principles of Software Development and Design
  • Artificial Intelligence - Introduction to AI and Machine Learning
  • Computer Security - Basics of Cybersecurity and Cryptography
  • Project - Capstone Project to demonstrate practical skills
